## 💡 開発の背景

日々の React Native 開発の中で、

> 「画面のキャプチャから素早くAIへ状況を伝えたい」

という課題から生まれました。

Swift のネイティブな操作性が必要だったため、AIとのペアプログラミングを通じて実装した実験的プロジェクトです。

---

## 🚀 主な機能

### ⚡ 即時キャプチャ

画面上の気になる部分を素早く選択し、そのまま解析できます。

### 🤖 AIによる画像解析

Gemini API を利用して、スクリーンショットの内容を解析します。

### 🔒 プライバシー重視

* ユーザー自身の Gemini API Key を使用
* API Key はローカル環境（UserDefaults）にのみ保存
* 開発者を含む第三者へ送信されることはありません

---

## ⚙️ セットアップ

### 1. リポジトリを取得

```bash
git clone <repository-url>
```

または ZIP ファイルをダウンロードしてください。

### 2. Xcodeで開く

プロジェクトを Xcode で開きます。

### 3. API Key を設定

1. アプリを起動
2. 右上の ⚙️ をクリック
3. 「APIキーを設定」を選択
4. Google AI Studio で取得した Gemini API Key を入力

設定内容は自動保存されます。

## ⚠️ 注意事項

### 実験的プロジェクトについて

* 本アプリは個人開発による実験的プロジェクトです
* コードの大部分はAI支援のもとで生成されています
* 予期しない動作やバグが発生する可能性があります

### 🗑️ キャッシュ削除機能について

**重要**

キャッシュ削除機能は、

```text
ScreenLens_
```

で始まる一時ファイルを削除対象とします。

そのため、重要なスクリーンショットは事前に別の場所へ保存することを推奨します。
