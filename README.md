#ScreenLens
スクリーンショットを即座に Gemini で解析する、macOS 用の生産性向上ツールです。

##💡 背景
日々の React Native 開発において、「画面のキャプチャから素早く AI に意図や問題を伝えたい」というニーズから生まれました。Swift のネイティブな操作性が必要だったため、AI とのペアプログラミングを通じて実装された実験的プロジェクトです。

##🚀 特徴
即時キャプチャ: 画面上の気になる部分を素早く解析。

AI による高度な解析: Gemini API を活用し、スクリーンショットの内容を正確に読み取ります。

プライバシー優先: 本アプリは、ユーザー様ご自身の Gemini API Key を使用する設計です。設定したキーはローカル環境（UserDefaults）にのみ安全に保存され、外部への送信は一切行われません。

##⚠️ 免責事項
本アプリは個人開発の実験的プロジェクトです。

コードの大部分は AI の支援を受けて生成されています。

予期せぬ動作やバグが含まれる可能性がありますが、ご理解の上ご利用ください。

[🛑最重要！！]
**キャッシュ削除機能について:** 本アプリのキャッシュ削除機能は `ScreenLens_` で始まるプレフィックスを持つ一時ファイルを対象とします。該当する一時ファイルは自動的に削除される可能性があるため、重要なスクリーンショットは別途保存しておくことを推奨します。

##⚙️ セットアップ
本リポジトリをクローンまたはダウンロードして Xcode で開きます。

アプリを起動します。

右上のギアアイコン（⚙️）から「APIキーを設定」を開き、Google AI Studio で取得したご自身の Gemini API Key を入力してください。

設定は自動的に保存されます。
